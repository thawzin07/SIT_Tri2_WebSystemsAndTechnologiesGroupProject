# SIT Tri2 Web Systems and Technologies Group Project

Group project repository for `INF1005 - Web Systems and Technologies`.

## Repository Contents

- `INF1005 Project Guide.pdf`: Official project guide provided for the module.
- Source code and project deliverables will be added by the team.

## Suggested Project Stack

- Frontend: HTML5, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL
- Deployment: Google Cloud LAMP server

## Team Workflow

1. Create feature branches for each task.
2. Open pull requests for review before merging into `main`.
3. Keep commits small and descriptive.
4. Do not commit credentials, API keys, or `.env` files.
